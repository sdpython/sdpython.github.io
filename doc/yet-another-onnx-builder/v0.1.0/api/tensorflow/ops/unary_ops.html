yobx.tensorflow.ops.unary_ops¶

Exp, Log, Log1p, Expm1

Trigonometric¶

Cos, Sin, Tan, Acos, Asin, Atan

Hyperbolic¶

Cosh, Sinh, Acosh, Asinh, Atanh

Special functions¶

Erf, Erfc, Softplus, Softsign

Rounding / Magnitude¶

Abs, Neg, Sign, Floor, Ceil, Round, Rint

Square-root family¶

Sqrt, Rsqrt, Square, Reciprocal
