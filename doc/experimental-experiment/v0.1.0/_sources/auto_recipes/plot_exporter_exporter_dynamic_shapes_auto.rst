Use DYNAMIC or AUTO when dynamic shapes has constraints
=======================================================

Settings the dynamic shapes is not always easy.
Here are a few tricks to make it work.

    dz = torch.export.Dim("dz")
    try:
        ep = torch.export.export(
            model,
            (x, y, z),
            dynamic_shapes={
                "x": {0: batch, 1: dx},
                "y": {0: batch, 1: dy},
                "z": {0: batch, 1: dz},
