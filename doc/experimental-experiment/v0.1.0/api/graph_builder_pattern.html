RotaryConcatPartPattern#

class experimental_experiment.torch_exp.patterns.RotaryConcatPartPattern[source]#

Optimizes the following sequence.

    opset: domain='' version=18
    input: name='X' type=dtype('float32') shape=['a', 'b', 'c', 'd']
    init: name='shape' type=dtype('int64') shape=(4,) -- array([   2,    2, 1024,  256])
    init: name='c0' type=dtype('int64') shape=(1,) -- array([0])
    init: name='c256' type=dtype('int64') shape=(1,) -- array([256])
    init: name='c512' type=dtype('int64') shape=(1,) -- array([512])
    init: name='c3' type=dtype('int64') shape=(1,) -- array([3])
    ConstantOfShape(shape) -> C1
    ConstantOfShape(shape) -> C2
    Slice(X, c256, c512, c3) -> S1
      Concat(C1, S1, axis=3) -> P1
    Slice(X, c0, c256, c3) -> r4_0
      Neg(r4_0) -> S2
      Concat(C1, S2, axis=3) -> P2
        Add(P1, P2) -> Y
    output: name='Y' type=dtype('float32') shape=['a', 'b', 'c', 'd']
