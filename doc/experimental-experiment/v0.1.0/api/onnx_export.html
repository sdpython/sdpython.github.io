to_onnx#

experimental_experiment.torch_exp.onnx_export.to_onnx(mod: torch.nn.Module | torch.fx.GraphModule, args: Sequence[torch.Tensor], input_names: Sequence[str] | None = None, target_opset: int | Dict[str, int] = 18, as_function: bool = False, remove_unused: bool = False, constant_folding: bool = False, verbose: int = 0) → ModelProto[source]#

Exports a torch model into ONNX using dynamo export.

Parameters:

  • mod – torch module

  • args – input arguments

  • input_names – input names

  • target_opset – targeted opset or targeted opsets as a dictionary

  • as_function – export as a ModelProto or a FunctionProto

  • remove_unused – if True, remove unused nodes

  • constant_folding – if True, fold constants, constants are detected while converting the model

  • verbose – verbosity level

Returns:

onnx model
