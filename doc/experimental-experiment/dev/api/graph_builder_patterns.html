Optimization Patterns¶

Onnx (default) Patterns¶

class experimental_experiment.xoptim.patterns.CastPattern(verbose: int = 0, priority: int = 0)[source]¶

Checks that a Cast is really needed.

class experimental_experiment.xoptim.patterns.CastCastBinaryPattern(verbose: int = 0, priority: int = 1)[source]¶

Moves two cast operators beyond a binary operator The cast must cast from a float type to another float type.

class experimental_experiment.xoptim.patterns.CastOpCastPattern(verbose: int = 0, priority: int = 1)[source]¶

Removes two cast surrounding another operator.

class experimental_experiment.xoptim.patterns.ComputationCastOpCastPattern(verbose: int = 0, priority: int = 1)[source]¶

Changes the computation type to make it faster if one of the inputs was just casted before.

class experimental_experiment.xoptim.patterns.DivByMulScalarPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces Div/Scalar by Mul*Scalar.

class experimental_experiment.xoptim.patterns.ExpandPattern(verbose: int = 0, priority: int = 0)[source]¶

Checks that a Expand is really needed.

class experimental_experiment.xoptim.patterns.ExpandBroadcastPattern(verbose: int = 0, priority: int = 1)[source]¶

Checks that a Expand is really needed before an element wise operator. The objective is to save one allocation and let the next operator do the expansion by broadcasting one input.

class experimental_experiment.xoptim.patterns.ExpandSwapPattern(verbose: int = 0, priority: int = 1)[source]¶

Tries to move a node Expand forward in the graph. Expand + Exp can be changed into Exp + Expand. Then Exp applies on a tensor of a smaller or equal size.

class experimental_experiment.xoptim.patterns.IdentityPattern(verbose: int = 0, priority: int = 0)[source]¶

Replaces operator such as Div(X, 1), Mul(X, 1), Add(X, 0), Sub(X, 0), Transpose(X, [0, 1, 2, …]) into identity nodes.

class experimental_experiment.xoptim.patterns.MatMulReshape2Of3Pattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces the reshapes around a matmul It can be 3 or 2 out of 3. It is similar to experimental_experiment.xoptim.patterns.onnx_reshape.Reshape2Of3Pattern.

class experimental_experiment.xoptim.patterns.MulMulMulScalarPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces the sequence {Div | Mul} and {Div | Mul} + {Div | Mul} with {Div | Mul} Mul.

class experimental_experiment.xoptim.patterns.ReduceReshapePattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces the sequence Reduce* Reshape if reshape is only introduces to deal with a dimension kept because keepdim=1.

class experimental_experiment.xoptim.patterns.ReduceSumNormalizePattern(verbose: int = 0, priority: int = 1)[source]¶

Nodes equivalent to a reduction.

class experimental_experiment.xoptim.patterns.ReshapeMatMulReshapePattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces the sequence Reshape, Matmul, Reshape by Matmul.

class experimental_experiment.xoptim.patterns.Reshape2Of3Pattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces the reshapes around element-wise operators. It can be 3 or 2 out of 3.

class experimental_experiment.xoptim.patterns.ReshapeReshapeBinaryPattern(verbose: int = 0, priority: int = 1)[source]¶

Moves two reshape operators beyond a binary operator if it is possible.

class experimental_experiment.xoptim.patterns.ReshapeReshapePattern(verbose: int = 0, priority: int = 0)[source]¶

Replaces the sequence Reshape, Reshape by Reshape.

class experimental_experiment.xoptim.patterns.RotaryConcatPartPattern(verbose: int = 0, priority: int = 1)[source]¶

Optimizes the following pattern

(Source code, png, hires.png, pdf)

class experimental_experiment.xoptim.patterns.SameChildrenPattern(verbose: int = 0, priority: int = 0)[source]¶

Checks that a Cast is really needed.

class experimental_experiment.xoptim.patterns.SlicesSplitPattern(verbose: int = 0, priority: int = 1)[source]¶

Detects multiple slices into a split.

class experimental_experiment.xoptim.patterns.Sub1MulPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces the sequence (1 - X) x Y by Y - X x Y to avoid the creation of a constant in the graph. x means element wise multiplication.

class experimental_experiment.xoptim.patterns.SwitchOrderBinaryPattern(verbose: int = 0, priority: int = 1)[source]¶

If it makes sense, switches the order of two multiplications or two addtions if the broadcasting reduces one operator to a an insignificant number.

class experimental_experiment.xoptim.patterns.TransposeMatMulPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces the sequence Transpose, Matmul or Gemm into Gemm

class experimental_experiment.xoptim.patterns.TransposeReshapeMatMulPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces the sequence Transpose, Reshape, Matmul into Reshape, Transpose, Matmul if possible. Another optimizer will optimizes this sequence by using Gemm or better.

class experimental_experiment.xoptim.patterns.TransposeTransposePattern(verbose: int = 0, priority: int = 0)[source]¶

Removes two consecutive transpose if the second one put the tensor in origin shape.

class experimental_experiment.xoptim.patterns.UnsqueezeEqualPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces the sequence R -> Equal -> Unsqueeze, R -> Unsqueeze, into R -> Unsqueeze -> Equal.

class experimental_experiment.xoptim.patterns.UnsqueezeUnsqueezePattern(verbose: int = 0, priority: int = 0)[source]¶

Replaces the sequence Unsqueeze, Unsqueeze by Unsqueeze.

Custom Op Patterns¶

class experimental_experiment.xoptim.patterns_fix.add_reduction_scatter_nd.AddReductionScatterND(verbose: int = 0, priority: int = 1)[source]¶

Replaces ConstantOfShape + ScatterND with ScatterNDOfShape (com.domain).

class experimental_experiment.xoptim.patterns_exp.binary_operators.AddAddMulMulPattern(verbose: int = 0, priority: int = 3, broadcast: bool = False)[source]¶

Replaces Add + Add by AddAdd or Mul + Mul by MulMul if they operate on the same shape.

Parameters:

broadcast – allow broadcast on the first dimensions.

class experimental_experiment.xoptim.patterns_exp.binary_operators.AddMulPattern(verbose: int = 0, priority: int = 3, broadcast: bool = False)[source]¶

Replaces Add + Mul by AddMul or Mul + Add by MulAdd if they operate on the same shape.

Parameters:

broadcast – allow broadcast on the first dimensions.

class experimental_experiment.xoptim.patterns_exp.binary_operators.AddMulBroadcastPattern(verbose: int = 0, priority: int = 4, broadcast: bool = True)[source]¶

Replaces Add + Mul by AddMul or Mul + Add by MulAdd if they operate on the same shape.

Parameters:

broadcast – allow broadcast on the first dimensions.

class experimental_experiment.xoptim.patterns_exp.binary_operators.AddMulSharedInputPattern(verbose: int = 0, priority: int = 3, broadcast: bool = False)[source]¶

Replaces Add(A, B) and Add(A, C) by AddSharedInput(A, B, C) if they operate on the same shape. Does the same for operator Mul.

Parameters:

broadcast – allow broadcast on the first dimensions.

class experimental_experiment.xoptim.patterns_exp.binary_operators.AddMulSharedInputBroadcastPattern(verbose: int = 0, priority: int = 4, broadcast: bool = True)[source]¶

Replaces Add(A, B) and Add(A, C) by AddSharedInput(A, B, C) if they operate on the same shape. Does the same for operator Mul.

Parameters:

broadcast – allow broadcast on the first dimensions.

class experimental_experiment.xoptim.patterns_exp.binary_operators.AddMulTransposePattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces (AddMul|MulAdd) + Transpose by (AddMul|MulAdd)(., transposeMiddle=1) if it is possible.

class experimental_experiment.xoptim.patterns_exp.constant_of_shape_scatter_nd.ConstantOfShapeScatterNDPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces ConstantOfShape + ScatterND with ScatterNDOfShape.

class experimental_experiment.xoptim.patterns_exp.constant_of_shape_scatter_nd.MaskedShapeScatterNDPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces Equal, Where, ScatterNDOfShape by MaskedScatterNDOfShape.

class experimental_experiment.xoptim.patterns_exp.binary_operators.MulSigmoidPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces Mul + Sigmoid by MulSigmoid if they operate on the same input.

class experimental_experiment.xoptim.patterns_exp.binary_operators.NegXplus1Pattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces 1 - X by NegXplus1 if they operate on the same input.

class experimental_experiment.xoptim.patterns_exp.where_replace.ReplaceZeroPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces Where(bool(X), value, X) into ReplaceZero(X, by=by).

class experimental_experiment.xoptim.patterns_exp.simple_rotary.SimpleRotaryPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces ConstantOfShape + ScatterND with ScatterNDOfShape (com.domain).

class experimental_experiment.xoptim.patterns_exp.binary_operators.SubMulPattern(verbose: int = 0, priority: int = 3, broadcast: bool = False)[source]¶

Replaces Sub + Mul by AddMul or Mul + Add by MulAdd if they operate on the same shape.

Parameters:

broadcast – allow broadcast on the first dimensions.

class experimental_experiment.xoptim.patterns_exp.binary_operators.SubMulBroadcastPattern(verbose: int = 0, priority: int = 4, broadcast: bool = True)[source]¶

Replaces Add + Mul by AddMul or Mul + Add by MulAdd if they operate on the same shape.

Parameters:

broadcast – allow broadcast on the first dimensions.

class experimental_experiment.xoptim.patterns_exp.unary_operators.TransposeCastPattern(verbose: int = 0, priority: int = 3)[source]¶

Replaces Cast + Transpose or Transpose + Cast into Transpose2DCast16 or Transpose2DCastFP32 depending on the output type.

class experimental_experiment.xoptim.patterns_exp.constants.TriMatrixPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces a sequence of nodes creating a triangular matrix with operator TriMatrix(…).

Ort Patterns¶

class experimental_experiment.xoptim.patterns_ort.fused_matmul.FusedMatMulPattern(verbose: int = 0, priority: int = 2)[source]¶

Replaces the sequence Transpose, Matmul into FusedMatMul.

class experimental_experiment.xoptim.patterns_ort.fused_matmul.FusedMatMulx2Pattern(verbose: int = 0, priority: int = 3)[source]¶

Replaces the sequence Div by a scalar consumed by two FusedMatMul.

class experimental_experiment.xoptim.patterns_ort.fused_matmul.FusedMatMulTransposePattern(verbose: int = 0, priority: int = 3)[source]¶

Replaces the sequence (Fused)Matmul(A,B) + Transpose into FusedMatMul(B.T, A.T).

class experimental_experiment.xoptim.patterns_ort.gather_grad.GatherGradPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces ConstantOfShape + ScatterND with GatherGrad (com.domain).

class experimental_experiment.xoptim.patterns_ort.simplified_layer_normalization.SimplifiedLayerNormalizationPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces the sequence Transpose, Matmul into FusedMatMul.

class experimental_experiment.xoptim.patterns_ort.activation_grad.SoftmaxGradPattern(verbose: int = 0, priority: int = 1)[source]¶

Replaces the sequence Mul, ReduceSum, Mul, Sub by SoftmaxGrad
