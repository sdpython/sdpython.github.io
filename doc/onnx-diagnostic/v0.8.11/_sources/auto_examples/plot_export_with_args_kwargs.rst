==========================================
Dynamic Shapes for ``*args``, ``**kwargs``
==========================================

Quick tour of dynamic shapes.
We first look at examples playing positional and names parameters
to understand how :func:`torch.export.export` works.

====

    class Model(torch.nn.Module):
